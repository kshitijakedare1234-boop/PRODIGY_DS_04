## Task 04 - Data Science Internship at Prodigy Infotech

### Objective
Perform sentiment analysis on social media data to understand public opinion and attitudes towards specific topics, brands, or events.

### Dataset
Twitter Sentiment Dataset

### Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- NLTK / TextBlob

### Steps Performed
1. Loaded the dataset using pandas.
2. Cleaned the text data by removing punctuation, links, and special characters.
3. Processed text using Natural Language Processing techniques.
4. Classified tweets into positive, negative, or neutral sentiments.
5. Visualized the sentiment distribution using charts.

### Output
- Sentiment classification of tweets
- Visualization of sentiment distribution
- Insights into public opinion trends

### Conclusion
The sentiment analysis model helps understand how people feel about a particular topic by analyzing social media text data.
